# kollab-cli skill

Open-source agent skill that teaches an AI agent how to operate a Kollab
workspace through the official `kollab` command-line tool: auth, spaces,
projects, tasks, skills, artifacts, timers, bots, MCP servers, memory,
feedback, and prompt-gallery search.

## Install

```bash
npx skills add KollabTeam/Kollab@kollab-cli
```

## Prerequisite

The skill drives the `kollab` CLI; install the binary and sign in:

```bash
npm install -g kollab
kollab login
```
